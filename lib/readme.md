<!-- 购物车 -->
购物车功能包含商品的增删改查、动态计算总价、结算功能,
可通过商品详情页和列表页底部悬浮按钮进入。


实现购物车功能，包含以下功能：
1. 结算：查看总价并结算。
2. 购物车入口：详情页和列表页展示。
3. 添加商品：详情页点击“添加到购物车”，添加商品信息并打开购物车页面。
4. 购物车管理：查看、增删改查商品，动态计算总价。
5. 结算提示：结算成功后提示并清空购物车。
